# 🌿 AI-ve — 새록 (Saerok)

> **자연이 싹트는 순간, 당신 곁에서.**

공공 생태 데이터와 AI 에이전트를 연결해 **위치·연령 맞춤형 생태 교육**을 제공하는 서비스.
검색하지 않아도, 예약하지 않아도 — 지금 서 있는 그 자리에서 스스로 찾아옵니다.

```
GPS 진입 감지  →  연령 맞춤 해설 자동 생성  →  퀴즈 · 배지
```

---

### 🎯 왜 만들었나

국립공원 탐방객 **연 4,300만 명** 중  
현장에서 생태 해설을 받는 사람은 **0.5%**.  
나머지 99.5%를 위한 서비스입니다.

같은 장소에 서도 5세 아이와 성인이 받는 콘텐츠는 하나뿐인 안내판.  
새록은 연령별로 다른 언어와 형식으로, 추가 비용 없이 즉시 생태 해설을 제공합니다.

---

### 🤖 핵심 기능

| 기능 | 설명 |
|------|------|
| **연령 맞춤 해설** | 미취학·초등·청소년·성인 4단계, 같은 장소 다른 언어 |
| **Geofencing 자동 트리거** | 생태·자연도 1등급 구역 진입 시 백그라운드 푸시 |
| **RAG 기반 퀴즈·배지** | 국가생물종목록 API + 에코뱅크 데이터 기반 신뢰성 확보 |
| **학습 포트폴리오** | 방문 이력·배지·점수 누적, 월별·계절별 통계 |
